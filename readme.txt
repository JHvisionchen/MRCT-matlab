Quickstart

1. Extract vlfeat-0.9.20.rar into the folder mrctrack\vlfeat-0.9.20
1. Extract toolbox-master.rar into the folder mrctrack\utility\toolbox-master

2. The tracker is prepared to run on all 50 videos of the OTB2013
 [2]. For that, it must know where they are located. You can
   change the default location 'base_path' in 'run_tracker.m.

4. Execute 'demos.m' without parameters to test MRCTracking on OTB2013.

__________

For the actual tracking code, check out the 'tracker_fea' function.
